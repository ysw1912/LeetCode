### 338. Counting Bits

-----

&emsp;&emsp;给定一个非负整数 num。对于 [0, num] 范围中的每个数字，计算其二进制数中的 1 的数目并将它们作为数组返回。

-----

#### 正解一  时间 O(n)

&emsp;&emsp;i & (i - 1) 可以移除 i 的最后一位 1，则 dp[i] = dp[i & (i - 1)] + 1。

#### 正解二  时间 O(n)

&emsp;&emsp;i >> 1 可以移除最低位，如果最低位为 0，则 dp[i] = dp[i >> 1]；如果最低位为 1，则 dp[i] = dp[i >> 1] + 1。
